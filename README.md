# Practica_POO
------
## Descripción
Proyecto en Java donde se aplican conceptos básicos de programación orientada a objetos.
---
## Temas aplicados
- Type Casting
- Packages
- Enums
- Initializer Block
- Pass by Value
- Exception Handling
- Access Specifiers
- Encapsulation
- Basics of OOP
- Method Overloading and Overriding
